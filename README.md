## 매일 갱신 규칙

1. 새 날짜의 `archive/YYYY-MM-DD/index.html`과 `source/YYYY-MM-DD.md`를 추가합니다.
2. `archive/index.html` 웹 목록과 `archive/README.md` 저장소 목록에 새 날짜를 추가합니다.
3. 루트 `README.md`의 날짜별 발행 목록을 최신순으로 갱신합니다.
4. 마지막으로 루트 `index.html`을 최신호로 교체합니다.
5. 기존 날짜 보관본은 덮어쓰지 않습니다.
