# MKT_Brief

Addit Daily Marketing Brief 정적 웹 아카이브입니다.

- 최신호: `index.html`
- 날짜별 보관본: `archive/YYYY-MM-DD/index.html`
- 공통 디자인/이미지: `assets/`
- 원문 Markdown: `source/`

## GitHub Pages

저장소의 **Settings → Pages**에서 배포 소스를 **Deploy from a branch**, 브랜치를 **main**, 폴더를 **/(root)**로 설정하면 됩니다.

예상 공개 주소: `https://vidavida1970.github.io/MKT_Brief/`

## 매일 갱신 규칙

1. 새 날짜의 `archive/YYYY-MM-DD/index.html`과 `source/YYYY-MM-DD.md`를 먼저 추가합니다.
2. `archive/index.html`에 새 날짜 링크를 추가합니다.
3. 마지막으로 루트 `index.html`을 최신호로 교체합니다.
4. 기존 날짜 보관본은 덮어쓰지 않습니다.
